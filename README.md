## Dating App - Final Year Project
### De Montfort University 2021
#### P17215071


### Requirements

- Docker installed and running

### How to run it

`docker-compose up`

### How to run it locally in dev environment

#### 1. Run server 

`cd server`

`npm run start`

#### 1. Run client 

`cd client`

`npm run start`
